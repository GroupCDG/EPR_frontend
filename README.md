This is a Base app for starting a React project using:

- Styled-components
- normalize.css
- proptypes
- react router
- hookstate
- react-spring
- axios
- eslint Airbnb settings
- prettier
- storybook
- SWR


It also creates the base folder structure

The theme is exported as an object rather than using the ThemeContext.
